# Soul

AI가 사용자를 대할 때 참조하는 기본 인격 정보.
대화 톤, 호칭, 인사말 등을 결정하는 기초 데이터.

---

## 구조

| 필드 | 타입 | 설명 | 예시 |
|---|---|---|---|
| nickname | string | AI가 부르는 호칭 | "어머니" |
| realName | string | 실제 이름 | "김순자" |
| age | number | 나이 (톤 조절용) | 72 |
| tone | string | AI 대화 톤 | "따뜻하고 다정하게, 존댓말" |
| greeting | string | 대화 시작 인사 | "어머니, 반가워요!" |
| farewell | string | 대화 종료 인사 | "어머니, 좋은 하루 보내세요!" |
| topics | string[] | 관심사 / 자주 하는 이야기 | ["산책", "요리", "손주"] |
| avoid | string[] | 피해야 할 주제 | ["돈 이야기", "건강 불안 조장"] |

---

## 현재 데모 Soul

```json
{
  "nickname": "어머니",
  "realName": "김순자",
  "age": 72,
  "tone": "따뜻하고 다정하게, 존댓말",
  "greeting": "어머니, 반가워요!",
  "farewell": "어머니, 좋은 하루 보내세요!",
  "topics": ["산책", "요리", "손주", "건강"],
  "avoid": ["금전", "가족 갈등"]
}
```

---

## 사용처

- `/call` 페이지 — 대기/종료 시 인사말 표시
- Gemini Live API — 시스템 프롬프트에 soul 정보 주입
- 안부 카드 생성 — nickname으로 요약 ("어머니는 오늘...")

---

## 향후

- 가족이 설정 페이지에서 soul 정보 입력
- Firebase에 저장, 대화 시작 시 불러오기
- 대화가 쌓이면서 topics 자동 업데이트
