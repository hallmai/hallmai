# 할마이 MVP 기획서
> 시니어 AI 말벗 · Gemini Live Agent Challenge 출품작
> 카테고리: Live Agents

---

## 한 줄 요약

버튼 하나 누르면 AI가 듣고, 공감하고, 가족에게 안부를 정리해준다.

---

## 플랫폼

모바일 웹 (네이티브 앱 불필요)

챌린지 규정상 플랫폼 제한 없음. 실제 출품작들도 Cloud Run에 올린 웹앱 형태. WebRTC로 모바일 브라우저에서 마이크 접근, Gemini Live API와 WebSocket 연결.

---

## 데모에서 보여줄 것 (3가지)

| # | 기능 | 핵심 기술 |
|---|---|---|
| 1 | 버튼 하나 → 실시간 음성 대화 시작 | Gemini Live API |
| 2 | 감정 감지 → 공감 모드 자동 전환 | Affective Dialog |
| 3 | 대화 종료 후 안부 카드 자동 생성 | Cloud Run · Firebase |

---

## MVP 스코프

### 만든다

- 어머니 음성대화 화면 (`/call`) — 큰 버튼 하나, 누르면 AI와 대화
- 실시간 음성 대화 — Gemini Live API 기반, 인터럽트 지원
- 감정 감지 + 모드 자동 전환 (공감 <-> 정보)
- 날씨 / 간단한 생활 정보 function calling
- 대화 종료 후 안부 카드 자동 생성 & Firebase 저장
- 가족 대시보드 (`/`) — 안부 카드 열람 (읽기 전용)
- 음성 브리핑 — 어제의 안부 요약을 TTS로 듣기
- AI 선제 연락 — 매일 정해진 시간에 AI가 어머니에게 대화 시작
- Google Cloud Run 배포

### 안 만든다

- 가족 → 어머니 답장 / 전화 / 사진 전송
- 에이전트 루프 (가족 액션 기반) → AI 자동 루프로 대체
- 공유 정책 설정 UI

---

## 안부 카드 스펙

대화 종료 후 자동 생성. 고정 필드로 구성.

| 필드 | 내용 | 비고 |
|---|---|---|
| 오늘 기분 | 좋음 / 보통 / 우울 | 대화 톤 기반 추정 |
| 한 줄 요약 | 오늘 주로 나눈 이야기 | 최대 2줄 |
| 주의 신호 | 외로움 / 통증 / 수면 언급 여부 | 없으면 표시 안 함 |

보안 규칙: 건강 세부 내용 · 가족 갈등 · 금전 관련 발화는 카드에 자동 포함 금지.

## 기술 스택

| 기술 | 역할 | 비고 |
|---|---|---|
| Gemini Live API | 실시간 음성 대화, 감정 감지, affective dialog | 심사 필수 요건 |
| Google Cloud Run | 서버리스 백엔드 배포 | 심사 필수 요건 |
| Google GenAI SDK | Gemini 모델 연동, function calling | 심사 필수 요건 |
| Firebase | 안부 카드 저장, 실시간 동기화 | |
| nestjs | WebSocket 서버, 오디오 스트리밍 | |
| WebRTC | 모바일 브라우저 실시간 오디오 스트림 | |
| Next.js + TypeScript | 프론트엔드 (가족 대시보드 + 어머니 통화 화면) | |
| Web Speech API (TTS) | 음성 브리핑 (추후 Gemini 음성으로 대체 가능) | |
